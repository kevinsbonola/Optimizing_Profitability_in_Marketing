# Project Overview
Comprehensive analysis of marketing data and customer behavior for the ticketing platform "Showz". This project focuses on optimizing marketing spend by analyzing server logs, order history, and customer acquisition costs.

## Objective
To transform raw data into strategic recommendations for the marketing department, determining campaign profitability and Customer Lifetime Value (LTV).

## Skills & Tools

Python: Pandas, Matplotlib, Seaborn.

Business Intelligence: Cohort Analysis, LTV (Lifetime Value), ROMI (Return on Marketing Investment), and CAC (Customer Acquisition Cost).

Data Analysis: Log cleaning, conversion funnel analysis, and retention metrics.

## Key Findings & Results

Ad Source Efficiency: Evaluated marketing channel performance based on Customer Acquisition Cost (CAC).

Cohort Analysis: Identified key patterns regarding user registration, conversion timing, and retention rates.

Strategic Recommendations: Provided actionable data-driven recommendations for budget allocation to maximize ROMI.

# Conclusion & Business Impact

By analyzing user behavior across different acquisition sources, I was able to identify high-converting channels that were previously undervalued. My analysis provided the marketing team with clear evidence to reallocate budget towards more profitable platforms, directly impacting the company's ROMI.
